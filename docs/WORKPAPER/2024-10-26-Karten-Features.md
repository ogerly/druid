# Fokus: Karten-Features

- leaflet, open streetmap karte
- offline modus von anfang an bedenken
- ein gps punkt setzen zu können
- einen weg gehen und verschiedenen gps punkte für eine wegzeichnung auf der karte
- marker setzen können
- der online weg ist primär. der offline muss nur mit gedacht werden, kann aber in einem weiteren schritt später richtig ausgebaut werden.

## Fortschritt

- **Grund-Setup:** Leaflet-Karte initialisiert und in die Vue-Anwendung integriert.
- **GPS-Zentrierung:** Button und Logik zum Zentrieren der Karte auf die aktuelle Benutzerposition hinzugefügt.
- **Marker setzen:** Manuelles Setzen von Markern per Klick auf die Karte ermöglicht.
